## Support Schema


# Schema v1
`paypal`: A Paypal email\
`patreon`: A Patreon username\
`twitter`: A Twitter username\
`youtube`: A Youtube url, minus `https://youtube.com/`

# Schema v2
`paypal`: A Paypal email\
`patreon`: A Patreon username\
`ko-fi`: A Ko-Fi username\
`twitter`: A Twitter username\
`youtube`: A Youtube url, minus `https://youtube.com/`
